###  Introduce

<hr>

ㆍ사용자의 MBTI 별 선호도를 기반으로 맞춤 스타일링 상품을 판매하는 쇼핑 플랫폼 서비스입니다.
  검증된 코디네이터를 통해, 특정 MBTI 유형인 사람이 선호하는 이성의 스타일링을 제공함으로써
  보다 감각적이고 센스있는 스타일링 상품을 편리하게 구매할 수 있습니다.
    
</br>
